= Comparison between modules and DITA topics

A module is similar to a topic in Darwin Information Typing Architecture (DITA) topic. A module differs from a DITA-topic in the following aspects:

* Source language: Modules are written AsciiDoc, whereas DITA topics are written in xml.
* Structure: Although both module and DITA-topic are structured, their structure differs.
